2. Datos que recopilamos

Según el uso del sitio web, podemos tratar:

  • Datos identificativos y de contacto (nombre, email, teléfono) cuando el usuario los facilita.
  • Datos de navegación (IP, dispositivo, navegador, páginas visitadas), especialmente si se usan cookies/analítica.
  • Información enviada mediante formularios (consultas, solicitudes, mensajes).

3. Finalidades del tratamiento

Tratamos tus datos para:

  • Atender consultas y solicitudes realizadas a través de formularios o email.
  • Gestionar la relación con el usuario/cliente y prestar los servicios solicitados.
  • Enviar comunicaciones comerciales si has dado tu consentimiento [SI APLICA].
  • Mejorar la experiencia de navegación y la seguridad del sitio web.
  • Cumplir obligaciones legales.

4. Base jurídica

La legitimación para el tratamiento puede ser:

  • Consentimiento (formularios, newsletter, cookies no necesarias).
  • Ejecución de un contrato o medidas precontractuales (si se contrata o compra).
  • Interés legítimo (seguridad, prevención del fraude, analítica básica cuando proceda).
  • Cumplimiento legal (obligaciones fiscales/contables y otras aplicables).

5. Conservación de los datos

Los datos se conservarán durante:

  • El tiempo necesario para cumplir la finalidad para la que se recabaron.
  • Los plazos legales aplicables (fiscales, contables, responsabilidad).
  • Hasta que el usuario solicite su supresión o retire el consentimiento cuando corresponda.
